🤖 AI Data Analytics Platform

📌 Overview

AI Data Analytics Platform is an intelligent system that allows users to upload datasets, clean data, visualize insights, and generate AI-powered analysis automatically.

This project combines data processing, graph analysis, and large language models to provide meaningful insights and storytelling from raw data.

---

🚀 Features

- 📂 Upload CSV / Excel datasets
- 🧹 Data cleaning using Pandas & NumPy
- 📊 Visualization and graph analysis
- 🔗 Data storage using Neo4j
- 🤖 AI-powered insights generation
- 💬 Natural language query support
- 📈 Automated reports and storytelling

---

🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- Data Processing: Pandas, NumPy
- Database: Neo4j
- AI Integration: OpenRouter / LLM APIs

---

🔄 Workflow

1. Upload Data
2. Clean & Process Data
3. Store in Database
4. User Query
5. AI Analysis
6. Generate Insights & Reports

---

⚙️ Installation

git clone https://github.com/your-username/ai-data-analytics.git
cd ai-data-analytics
pip install -r requirements.txt
python app.py

---

▶️ Usage

- Open browser and go to: "http://localhost:5000"
- Upload dataset
- Ask questions in natural language
- View insights and visualizations
